"# SFU-Survival-Bot"
Project Escape SFU


Problem Description
  - Simon Fraser University is located on top of mountain
  - Every winter, campus closes without notice
  - Lack of centralized info program

Objective
  - Create a program that notifies students regarding following info:
    - Weather Status
    - Bus Status
    - Library Hours
    - Road Conditions
    - Ride Share
